# Antennas
# Antenna Design Simulations using HFSS & AEDT

This repository contains multiple antenna design and electromagnetic wave simulation projects created using ANSYS HFSS and ANSYS Electronics Desktop (AEDT).

## 📁 Repository Contents

| File Name | Description |
|------------|-------------|
| `EM_wave_generation.hfss` | Electromagnetic wave generation simulation |
| `board_side_array_antenna.aedt` | Board-side array antenna design |
| `circular_patch_antenna.hfss` | Circular patch antenna simulation |
| `folded_dipole_antenna.hfss` | Folded dipole antenna model |
| `half_wave_dipole_antenna.hfss` | Half-wave dipole antenna design |
| `horn_antenna.hfss` | Horn antenna simulation |
| `monopole_antenna.hfss` | Monopole antenna design |
| `rectangular_patch_antenna.hfss` | Rectangular microstrip patch antenna |
| `reflector_antenna.hfss` | Reflector antenna simulation |
| `yagiuda_antenna.hfss` | Yagi-Uda antenna design |
